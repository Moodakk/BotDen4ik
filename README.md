# BotDen4ik 🦷

**Telegram-бот для підготовки українського стоматолога до прийому чеських пацієнтів.**

> A Telegram bot that helps Ukrainian-speaking dentists communicate with Czech patients.

---

## Можливості / Features

| Розділ | Опис |
|--------|------|
| 🦷 **Словник термінів** | Чесько-українські стоматологічні терміни за категоріями (анатомія, діагнози, процедури, інструменти) |
| 🗣 **Скарги пацієнтів** | Типові чеські фрази з перекладом і фонетичною підказкою |
| 💬 **Діалог на прийомі** | Питання від вітання й анамнезу до рекомендацій після лікування |
| 🚨 **Невідкладні фрази** | Термінові фрази при ускладненнях та алергічних реакціях |
| 🧠 **Вікторина** | Тест зі 5 випадкових питань для самоперевірки |

## Структура проекту / Project structure

```
BotDen4ik/
├── bot.py                 # Головний файл бота
├── requirements.txt       # Python-залежності
├── data/
│   ├── __init__.py
│   ├── vocabulary.py      # Стоматологічний словник (ЧЕС↔УКР)
│   ├── phrases.py         # Скарги та відповіді пацієнтів
│   ├── dialogues.py       # Діалоги на прийомі
│   ├── emergency.py       # Невідкладні фрази
│   └── quiz.py            # Питання для вікторини
└── tests/
    └── test_bot.py        # Unit-тести
```

## Запуск / Running the bot

### 1. Встановіть залежності

```bash
pip install -r requirements.txt
```

### 2. Отримайте токен

Зареєструйте бота у [@BotFather](https://t.me/BotFather) і отримайте `TELEGRAM_TOKEN`.

### 3. Запустіть

```bash
TELEGRAM_TOKEN=<your_token> python bot.py
```
